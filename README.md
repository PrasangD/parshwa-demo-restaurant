# Saffron & Sage — Restaurant Demo

A single-page demo website for a fictional modern Indian fine-dining & fusion restaurant, **Saffron & Sage**, set in Dombivli West, Mumbai.

This is a **demo / portfolio piece by [Parshwaventures](https://www.parshwaventures.co.in)** — a web & IT firm in Dombivli, India. All content, prices, reviews, and contact details are fictional and for demonstration only.

## What's inside
- Sticky translucent header with smooth-scroll nav and a working mobile hamburger menu
- Full-viewport hero, About/Story, Signature Menu (with ₹ prices), image Gallery, Chef/Experience, Testimonials, and a Reservation/Contact section
- Styled reservation form with inline success message (no backend — validates and confirms client-side)
- Reveal-on-scroll animations, hover micro-interactions, responsive mobile-first layout
- Opening hours, address, and a map placeholder

## How to view
Just open `index.html` in any modern web browser. No build step, no server, no dependencies to install.

## Tech
- Vanilla **HTML5**, **CSS3**, and **JavaScript** — everything inline in one file
- Google Fonts (Playfair Display + Manrope) and Unsplash photos loaded via CDN
- No frameworks or external JS/CSS libraries
